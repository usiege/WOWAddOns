## 代码说明

code | Description
:=: | :=: 
dd.lua | 打断，驱散等小队/团队通报
dt.lua | 小地图简化，改方形; <br> 去除指南针，日历，地区文本等多余的东西
dzt.lua | 动作条简化，隐藏边框，简化艾泽里特经验条，<br>超出范围技能图标变红，隐藏系统自带微型菜单栏
lt.lua | 聊天框简化，去除多余的东西
tx.lua | 头像简化，头像改职业图标，姓名背景改透明，战斗数字隐藏
zdxl.lua | 自动卖垃圾自动修理
sj.lua | 聊天栏频道名称缩写
njd.lua | 人物装备栏显示耐久度
npc.lua | 隐藏npc对话框
ncshadow | 景深效果，输入/shadow 1-100调节
